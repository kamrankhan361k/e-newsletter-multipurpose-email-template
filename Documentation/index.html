Templates

The E-Newsletter comes with 7 different premade templates. There are 3 folders inside the tempaltes folder named 'HTML, 'Mailchimp' and 'Campaign Monitor. These templates contain all the layouts and you can easily edit or modify the content and images.
The templates follow a new and easy to edit structure with proper comments in place.

HTML Structure

  1. All E-Newsletter templates follow a unique html structure which has been tested on litmus to achieve maximum success with all the major email clients.
  2. Every module/block is nested inside a table and the same struture appliess for all available modules.
  3. All modules are marked with comments a comment is place at the start of a modules and is also marked at the end of the module.
